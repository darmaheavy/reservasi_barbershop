

A web-based barbershop reservation system developed using Laravel. This project allows customers to book barbershop services online while providing administrators with tools to manage bookings, schedules, services, and customers.

# Features

- Customer online booking
- Booking schedule management
- Operational hours management
- Admin Dashboard
- Service management
- Gallery management
- Booking validation
- WhatsApp notification
- Conflict booking detection
- Authentication & Authorization

##  Tech Stack

- Laravel 12
- PHP 8.3
- MySQL
- Tailwind CSS
- JavaScript
- Git

## Installation

```bash
git clone https://github.com/darmaheavy/reservasi_barbershop.git

cd reservasi_barbershop

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
